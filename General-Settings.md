## Maintenance Mode

**To activate Maintenance Mode:**

1.  Go to  **Admin Panel**
2.  Select  **Settings**  ->  **General**
3.  Set  **Maintenance Mode**  option to  **ON**
4.  Click  **Save**  button

While activated, anyone who tries to access will see following page.
